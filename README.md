## Usage

### npx

```sh
npx create-react-app myAppName --template klutch
```

### npm commands

```sh
# Starts the local dev server
npm start

# Build production ready app in the build folder
npm run build

# Analyze bundle size of the build
npm run analyze

# Clean the project folder
npm run clean
```
